# Care Labor Ledger

A lightweight SwiftUI app for experimenting with how care and emotional labor might be tracked in a simple personal ledger. This project runs entirely locally using SwiftUI with no external dependencies.

---

## Running the App Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/<YOUR-USERNAME>/<YOUR-REPO>.git

## Open the Project in Xcode

2. Open Xcode  
3. Go to **File → Open…**  
4. Select the project folder (the folder containing the `.swift` files and `Assets.xcassets`)

## Select a Simulator

5. Use the device selector in the toolbar (e.g., **iPhone 15 Pro**)

## Run the App

6. Press **⌘R** to build and launch the app
